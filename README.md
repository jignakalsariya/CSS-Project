# Spotify Web Player UI Clone

A professionally designed **Spotify Web Player UI Clone** built using **HTML5 and CSS3**.  
This project focuses on recreating a modern music streaming interface to practice real-world CSS layouts, responsiveness, and component-based UI design.

## Project Overview

This project replicates the visual structure of the **Spotify Web Player**.  
It is a **frontend UI project** with no backend or audio functionality, created purely to understand how large, production-level web interfaces are structured and styled using CSS.

The emphasis is on:
- Clean layout
- Visual hierarchy
- Responsive behavior
- Professional UI consistency

## Objectives

- Build a real product-inspired web interface
- Strengthen CSS Flexbox and layout skills
- Learn sticky and fixed positioning
- Design reusable UI components
- Create custom-styled range sliders
- Practice clean and maintainable CSS

## Features

- Sidebar navigation (Home, Search, Library)
- Library section with action cards
- Sticky top navigation bar
- Card-based content sections
- Fixed bottom music player
- Custom playback progress bar
- Volume control slider
- Responsive design with media queries

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Styling and layout
- **Font Awesome** – Icons
- **Google Fonts (Montserrat)** – Typography
